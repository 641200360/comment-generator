# 一键写评语小程序

## 项目说明
这是一个帮助老师快速生成个性化学生评语的网页工具，专注于期末评价生成。

## 项目结构
- index.html - 主页面
- styles.css - 样式文件
- script.js - 功能实现
- templates.js - 评语模板数据

## 页面结构
1. 主页面 (index.html)
   - 顶部导航栏
   - 主要功能区
     - 学生信息输入
     - 评价维度选择
     - 评语风格选择
     - 生成按钮
     - 评语展示区
   - 历史记录区
   - 帮助说明区

## 使用说明
1. 输入学生姓名
2. 选择评价维度（可多选）
3. 选择评语风格
4. 点击生成按钮
5. 在评语展示区查看结果
6. 可以复制或修改评语
